## 🚀 Collaborative Ventures

### 🌾 Cropix
CROPIX is an AI-driven agriculture platform designed to empower farmers. It provides precision recommendations for crops and fertilizers, detects plant diseases from images, and forecasts market prices and weather patterns.
- **Tech Stack:** ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white) ![Machine Learning](https://img.shields.io/badge/ML-FF6F00?style=for-the-badge&logo=tensorflow&logoColor=white)
- **Impact:** Helped farmers optimize yield and reduce resource wastage.

### 📄 Resumer
An intelligent resume analysis and optimization tool that uses NLP to help job seekers tailor their resumes to specific job descriptions, increasing their chances of passing ATS filters.
- **Tech Stack:** ![NLP](https://img.shields.io/badge/NLP-4B0082?style=for-the-badge&logo=google-cloud&logoColor=white) ![AI](https://img.shields.io/badge/AI-4169E1?style=for-the-badge&logo=openai&logoColor=white)
- **Goal:** Bridging the gap between talent and opportunity.

### 🍯 Agentic Honeypot
A sophisticated cybersecurity system that uses autonomous AI agents to simulate vulnerable environments. It lures attackers, analyzes their techniques in real-time, and generates threat intelligence reports.
- **Tech Stack:** ![Security](https://img.shields.io/badge/Security-DD0031?style=for-the-badge&logo=angular&logoColor=white) ![AI Agents](https://img.shields.io/badge/AI%20Agents-000000?style=for-the-badge&logo=langchain&logoColor=white)
- **Innovation:** Proactive threat detection using agentic orchestration.

### 🎣 Fish Pish
A real-time phishing detection system that analyzes URLs and email content to identify malicious intent. It uses a combination of heuristic analysis and machine learning models.
- **Tech Stack:** ![Cybersecurity](https://img.shields.io/badge/Cybersecurity-000000?style=for-the-badge&logo=hackthebox&logoColor=white) ![Scikit-learn](https://img.shields.io/badge/Scikit--learn-F7931E?style=for-the-badge&logo=scikit-learn&logoColor=white)
- **Status:** Developed during a 24-hour hackathon sprint.
